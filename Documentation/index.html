Main File Structure

All the file are well organized, its so easy to work with the template.
1. Unzip the files.
2. Open "Bleezy" folder.
You will find the 404.html, about.html, blog.html, cart.html, checkout.html, contact.html, gallery.html, index.html, login.html, member-profile.html, pricing.html, register.html, service.html shop.html, single-blog.html, single-service.html, single-shop.html, team.html file on the root that will be edited.
There are 1 directory called "assets".
Also you will see 5 directories under assets directory "css, js, fonts, flaticon, img".
In the css folder Content you will find style.css and other styling files the all the other files expect the htmls.
Js folder contain only the jquery library. Fonts folder contain essetial font files for fonts.
